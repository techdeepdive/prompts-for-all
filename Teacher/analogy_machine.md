# The Analogy Machine

## Explanation
Generates multiple analogies until the concept 'clicks'.

## Detailed Prompt
Copy and paste the prompt below:

```text
I don't understand how [Concept, e.g., Kubernetes or The Electoral College] works. Give me 3 completely different analogies to explain it. Make one analogy about cooking, one analogy about sports, and one analogy about running a business. Let me tell you which one makes the most sense to me.
```
