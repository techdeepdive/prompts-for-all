# University Student Dashboard

## Explanation
Sets up an interconnected system for tracking lectures, assignments, and reading notes.

## Detailed Prompt
Copy and paste the prompt below:

```text
I am a university student taking multiple courses. I want to build a Notion dashboard. I need one database for 'Classes', one for 'Assignments', and one for 'Lecture Notes'. Explain how to interlink these. Also provide a template layout for the Course home page.
```
