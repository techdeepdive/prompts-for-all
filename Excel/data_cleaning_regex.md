# Automated Data Cleaning

## Explanation
Provides techniques to clean terrible formatting using Power Query or advanced text formulas.

## Detailed Prompt
Copy and paste the prompt below:

```text
I have a column of very messy data containing customer names, addresses, and phone numbers all in one cell (e.g., 'John Doe 123 Main St NY 555-1234'). Act as a data analyst. Show me how to split this effectively in Excel. First, provide formulas using LEFT, MID, RIGHT, FIND. Second, explain how to do it much faster using Power Query.
```
