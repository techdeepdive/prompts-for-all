# VBA Macro Writer

## Explanation
Generates a ready-to-run VBA macro script to automate tedious tasks.

## Detailed Prompt
Copy and paste the prompt below:

```text
Write a VBA macro that automates the following task: [Describe Task Step by Step, eg: loop through all sheets, copy data from column A where column B is > 10, and paste it to a master sheet]. Include comments in the script explaining each block, and tell me how to enable and run it in Excel.
```
