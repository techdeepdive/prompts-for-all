# E-Commerce Product Description

## Explanation
Writes sensory-rich, conversion-focused descriptions for Shopify/Amazon items.

## Detailed Prompt
Copy and paste the prompt below:

```text
Act as an elite e-commerce copywriter. Write a 200-word product description for [Product Name, e.g., a hand-poured lavender soy candle]. Use sensory words to paint a picture of the experience of using the product. Break it up into a short evocative paragraph, followed by 4 bulletized features highlighting materials and origin.
```
