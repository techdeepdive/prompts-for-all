# Landing Page Copywriter

## Explanation
Generates the main elements of a SaaS or product landing page.

## Detailed Prompt
Copy and paste the prompt below:

```text
I am launching a new product called [Product Name], which helps [Target Audience] achieve [Desired Outcome] without [Pain Point]. Act as a conversion-rate-optimization copywriter. Generate the Hero Headline, Sub-headline, 3 benefit-driven bullet points, and the main Call-to-Action button text. Focus on clear, punchy benefits, not features.
```
