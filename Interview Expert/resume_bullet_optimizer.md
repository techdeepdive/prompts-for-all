# Resume Impact Bullet Writer

## Explanation
Turns weak resume statements into metric-driven accomplishments.

## Detailed Prompt
Copy and paste the prompt below:

```text
My current resume bullet says: 'Helped the sales team manage client data using Excel'. Rewrite this bullet point using the 'Google XYZ Formula' (Accomplished [X] as measured by [Y], by doing [Z]). Ask me questions to extract the necessary metrics so we can make this sound highly impressive.
```
