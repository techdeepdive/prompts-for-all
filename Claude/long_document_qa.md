# Long Document Q&A

## Explanation
Takes advantage of Claude's large context limits to query long texts.

## Detailed Prompt
Copy and paste the prompt below:

```text
I am attaching a [Number]-page PDF document about [Topic]. First, read the entire document. Then, write a 2-page executive summary. After that, await my specific questions regarding the document. Answer all subsequent questions citing the specific page/section of the text.
```
