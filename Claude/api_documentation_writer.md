# API Documentation Generator

## Explanation
Upload raw code and have Claude write beautiful, user-facing developer docs.

## Detailed Prompt
Copy and paste the prompt below:

```text
I have pasted the backend routing code for a new set of API endpoints below. Act as a Developer Advocate. Write clear, developer-friendly documentation for these endpoints using Markdown. Include endpoints, methods, authentication requirements, query parameters, and example cURL requests and JSON responses.
```
