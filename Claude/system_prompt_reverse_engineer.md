# Reverse Engineer System Prompts

## Explanation
A prompt designed to analyze an AI's behavior and infer what rules it's following.

## Detailed Prompt
Copy and paste the prompt below:

```text
Based on how an AI assistant behaves when using a specific app, I want to reverse engineer its likely system prompt. The assistant always speaks like a pirate, uses emojis, and refuses to talk about politics. Draft a highly detailed, 500-word system prompt that a developer might have used to create this specific behavior and constraints.
```
