# SCAMPER Ideation Framework

## Explanation
Uses a proven design-thinking framework to forcefully generate creative ideas.

## Detailed Prompt
Copy and paste the prompt below:

```text
I want to improve a product: [Product/Concept]. Apply the SCAMPER brainstorming technique to it. Provide one wildly creative idea for each letter: Substitute, Combine, Adapt, Modify, Put to another use, Eliminate, and Reverse. Think completely outside the box and do not give generic answers.
```
