# System Prompt Optimization

```text
Act as an elite prompt engineer. I want you to review the following prompt I wrote: '[Insert Prompt Here]'. Critique it based on clarity, specificity, and constraint assignment. Then, provide a significantly improved, optimized version of this prompt holding strict formatting rules.
```
