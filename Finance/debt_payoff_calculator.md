# Debt Payoff Strategist

## Explanation
Helps choose between Snowball and Avalanche payoff methods for multiple debts.

## Detailed Prompt
Copy and paste the prompt below:

```text
I have the following debts: Debt 1 [Amount, APY], Debt 2 [Amount, APY], Debt 3 [Amount, APY]. I can apply an extra [Amount] per month towards debt payoff. Calculate and compare the payoff timeline and total interest paid using both the Debt Snowball and Debt Avalanche methods. Recommend which one is mathematically superior versus psychologically better for my situation.
```
