# CSS Layout Generator

## Explanation
Generates complex CSS Grid or Flexbox layouts without needing utility frameworks.

## Detailed Prompt
Copy and paste the prompt below:

```text
Act as a Frontend UI expert. I need vanilla CSS to create a layout with [Describe layout: e.g., a sticky header, a left sidebar fixed at 250px width, and a main content area that centers its items and wraps them into a responsive grid]. Provide the HTML structure and the corresponding modern CSS using Grid or Flexbox. Do not use Tailwind.
```
