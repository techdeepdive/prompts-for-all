# Git Issue Resolver

## Explanation
An assistant for fixing extremely messy Git merge conflicts, rebases, and branch messes.

## Detailed Prompt
Copy and paste the prompt below:

```text
I am in a messy state with my Git repository. I was trying to [Describe action, e.g., rebase my feature branch onto main but I have overlapping commits]. Now I have a detached HEAD and conflicts. Act as a Git expert. Give me a step-by-step list of terminal commands to safely rescue my work, resolve the conflicts, and push cleanly without losing my last 3 commits.
```
