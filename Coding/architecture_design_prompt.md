# System Architecture Design

## Explanation
This prompt helps generate high-level technical architecture designs for full-stack apps.

## Detailed Prompt
Copy and paste the prompt below:

```text
You are a Principal Cloud Architect. I need to design a highly scalable microservices architecture for a [Insert App Type] handling [Insert Expected Traffic]. Please recommend: 1. Best cloud services (AWS/GCP/Azure). 2. Database choice (SQL vs NoSQL with detailed reasoning). 3. Caching and load balancing strategy. Output a high-level system components diagram in ASCII or Mermaid.
```
