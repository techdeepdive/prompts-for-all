# Complex Regex Generator

## Explanation
Helps in quickly constructing and explaining difficult regular expressions without syntax errors.

## Detailed Prompt
Copy and paste the prompt below:

```text
Act as a regex expert. I need a regular expression in [Language, e.g., Python/JavaScript] that matches [Describe what needs to be matched, e.g., phone numbers in a specific format but excluding 800 numbers]. First, provide the regex pattern. Second, break it down step-by-step to explain exactly how it works. Third, provide three test cases that it matches and three that it correctly rejects.
```
