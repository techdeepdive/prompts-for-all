# RESTful API Spec Generator

## Explanation
Generates a standardized OpenAPI/Swagger specification for a new endpoint.

## Detailed Prompt
Copy and paste the prompt below:

```text
I need to design a RESTful API endpoint for [Action, e.g., User Registration or Product checkout]. Provide the ideal URL route structure, the HTTP Method to use, the required JSON payload for the request body, and standard JSON responses for success (200/201), client errors (400), and unauthorized (401/403). Format the response as an OpenAPI 3.0 YAML snippet.
```
