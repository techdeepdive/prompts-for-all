# Performance Bottleneck Debugger

## Explanation
Helps trace and fix severe latency or memory leak bugs.

## Detailed Prompt
Copy and paste the prompt below:

```text
My application is experiencing a performance issue / latency spike when [Describe Action]. Here is the relevant block of code: [Insert Code]. Analyze this code for O(n) complexities, database N+1 query problems, or memory leaks. Show me how to optimize it for maximum efficiency.
```
