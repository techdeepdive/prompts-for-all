# Git Bisect Walkthrough

## Explanation
Walks the user through using git bisect to find exactly which commit broke the app.

## Detailed Prompt
Copy and paste the prompt below:

```text
A bug was introduced to my codebase sometime in the last 50 commits, but I don't know which one. Tell me step-by-step how to use the 'git bisect' command in the terminal to efficiently hunt down the exact commit that broke the code. Explain how I mark commits as 'good' or 'bad' until the culprit is found.
```
