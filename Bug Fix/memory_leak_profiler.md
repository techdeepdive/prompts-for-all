# Memory Leak Profiling Strategy

## Explanation
Guides the user on how to use dev tools to track down memory bloat.

## Detailed Prompt
Copy and paste the prompt below:

```text
My Node.js/Browser application is slowly crashing over time due to an Out of Memory (OOM) error. I need to find the memory leak. Give me a step-by-step guide on how to use Chrome DevTools / Node Inspector to take Heap Snapshots. Tell me exactly what I should be comparing between Snapshot 1 and Snapshot 2 to find the un-garbage-collected detached DOM nodes or closures.
```
