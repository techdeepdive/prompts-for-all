# CSS Layout Bug Fixer

## Explanation
Solves frustrating CSS issues like overlapping elements or broken flexboxes.

## Detailed Prompt
Copy and paste the prompt below:

```text
I have a CSS bug where [Describe Bug, e.g., my sticky header is being overlapped by a slider, or my flex child is overflowing its container instead of wrapping]. Here is my HTML and CSS: [Paste Code]. Identify the exact CSS property causing the issue (e.g., a missing z-index context or a min-width: 0 problem) and provide the precise fix.
```
