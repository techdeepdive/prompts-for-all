# Codebase Roasting

## Explanation
Have Grok mock your terrible code while simultaneously fixing it.

## Detailed Prompt
Copy and paste the prompt below:

```text
I wrote this Python code: [Paste Code]. I know it's probably terrible. Roast me as hard as you can for my variable names, my lack of efficiency, and my general coding incompetence. After you are done insulting me, provide the optimized, clean version of the code that a real engineer would write.
```
