# Feature Test Plan Creator

## Explanation
Creates a manual and automated testing strategy for a new feature.

## Detailed Prompt
Copy and paste the prompt below:

```text
We are launching a new feature: [Describe Feature]. Create a detailed Test Plan document. Include: 1. Scope of testing. 2. Manual Test Cases (step-by-step). 3. E2E Test Scenarios to automate. 4. Edge cases and potential security/regression risks to look out for.
```
