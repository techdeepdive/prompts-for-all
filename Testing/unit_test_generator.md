# Comprehensive Unit Test Generator

## Explanation
Generates robust unit tests and edge cases for a given function.

## Detailed Prompt
Copy and paste the prompt below:

```text
Act as a Senior QA Engineer. Here is a [Language] function: [Insert Code]. Generate a comprehensive test suite using [Testing Framework, e.g., Jest/PyTest]. Include standard cases, edge cases, null/undefined inputs, and boundary conditions. Explain what each test case is trying to break.
```
