# Test Mock Data Generator

## Explanation
Creates large sets of realistic JSON dummy data for testing databases or UIs.

## Detailed Prompt
Copy and paste the prompt below:

```text
I need realistic mock data to test a database pagination feature. Generate a JSON array containing 20 objects. Each object represents a 'User' and must contain: a UUID, a realistic sounding name, an email address that matches the name, a random date of birth between 1980 and 2000, and a boolean 'isActive' flag.
```
