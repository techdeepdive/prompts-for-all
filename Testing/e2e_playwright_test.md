# Playwright/Cypress E2E Script

## Explanation
Writes an end-to-end browser automation test script.

## Detailed Prompt
Copy and paste the prompt below:

```text
Act as an SDET (Software Development Engineer in Test). Write a [Playwright/Cypress] script in [TypeScript/JavaScript] that tests the following user flow: 1. Navigate to the login page. 2. Fill in valid credentials and submit. 3. Assert that the dashboard URL loads. 4. Verify that the user avatar is visible. Use best practices for selecting elements (e.g., data-testid).
```
