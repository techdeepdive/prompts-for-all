# WCAG Accessibility Review

## Explanation
Evaluates HTML for screen reader compatibility and accessibility compliance.

## Detailed Prompt
Copy and paste the prompt below:

```text
Act as a Web Accessibility (A11y) Expert. Review this block of HTML: [Paste HTML]. Identify any violations of WCAG 2.1 AA standards (e.g., missing ARIA labels, bad contrast constraints, non-semantic tags). Provide the corrected HTML and explain why the changes are necessary for screen readers.
```
