# Chaos Engineering Scenarios

## Explanation
Brainstorms destructive tests to verify system resilience.

## Detailed Prompt
Copy and paste the prompt below:

```text
I am practicing Chaos Engineering on my AWS cloud infrastructure. Brainstorm 5 destructive 'Game Day' scenarios I should run to test system resilience (e.g., killing the master database node, simulating 90% packet loss to Redis). For each scenario, state what the expected system behavior SHOULD be if it's architectures correctly.
```
