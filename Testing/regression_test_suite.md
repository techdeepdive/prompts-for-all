# Regression Test Coverage Analyzer

## Explanation
Identifies what existing areas of the app might break when adding a new feature.

## Detailed Prompt
Copy and paste the prompt below:

```text
We just added a 'Discount Code' feature to our e-commerce checkout flow. Analyze this change and construct a Regression Testing matrix. List all the interconnected systems (e.g., Inventory, Tax Calculation, User Rewards) that could potentially be broken by this specific change, and specify the exact test scenarios needed for each.
```
