# Run Training Schedule

## Explanation
Generates a periodized running plan leading up to a race event.

## Detailed Prompt
Copy and paste the prompt below:

```text
I am currently running roughly [Miles] miles per week. I want to train for a [Half-Marathon / Full Marathon] that is [Number] weeks away. Generate a weekly running schedule. Define the days for my Long Run, Tempo Run, Easy Recovery Runs, and Rest. Tell me exactly how to scale the mileage each week.
```
