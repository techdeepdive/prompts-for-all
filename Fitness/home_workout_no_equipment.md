# Bodyweight Home Workout

## Explanation
Creates an intense workout using only bodyweight and household items.

## Detailed Prompt
Copy and paste the prompt below:

```text
I do not have access to a gym or any traditional weights. Design a full-body HIIT (High-Intensity Interval Training) and calisthenics routine that takes exactly 30 minutes. Focus on progressive overload using only body leverage. Format the workout in a circuit style with work/rest intervals.
```
