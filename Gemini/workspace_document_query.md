# Google Workspace Integration Query

## Explanation
A prompt designed for Gemini when it's integrated with Google Docs/Drive.

## Detailed Prompt
Copy and paste the prompt below:

```text
Access my recent Google Docs. Find the document named '[Exact Doc Name]'. Read it entirely. Then, draft a highly professional email summarizing the 3 key action items discussed in that document, and address the email to my team telling them the deadlines. Output the email draft here.
```
